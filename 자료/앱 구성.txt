<시작화면>
시작하기 버튼을 누르면 로그인 화면으로 전환된다.

<로그인화면>
이름, 연락처를 입력한 후 로그인 버튼을 누르면
서버 DB에서 정보를 가져와 입력 정보와 비교한다.
입력한 정보와 일치하는 회원이 있을 경우에는 로그인 성공,
없을 경우에는 로그인 실패가 된다.

회원가입 버튼을 누르면 회원가입 화면으로 전환된다.

<회원가입 화면>
회원가입은 총 2단계로 나뉜다.
이름을 입력한 후 계속하기 버튼을 누르면 전화번호 입력을 할 수 있다.
전화번호를 입력을 한 후 회우너가입 완료 버튼을 누르면
서버 DB에 정보가 입력됨과 동시에 회원가입 성공이 된다.

<홈화면>
메뉴 버튼을 누르면 메뉴 화면으로 전환된다.

마이페이지 버튼을 누르면 마이페이지 화면으로 전환된다.

<메뉴 화면>
맨 위의 카테고리 버튼을 누르면 카테고리에 해당하는 메뉴들이 표시된다.
카테고리 버튼 중 음료 버튼을 누르면 하위 카테고리 버튼이 표시되고
버튼을 누르면 해당 메뉴들이 표시된다.'

<마이페이지 화면>
내 포인트 버튼을 누르면 서버 DB에서 회원 포인트 정보를 가져와 표시한다.

계정 정보 버튼을 누르면 서버 DB에서 회원 이름, 회원 전화번호 정보를 가져와 표시한다.
